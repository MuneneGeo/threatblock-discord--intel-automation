# Setup Guide

## Step 1: Import Workflow
- Open n8n
- Import JSON workflow file

## Step 2: Configure Discord Webhook
- Go to Discord server
- Channel settings → Integrations → Webhooks
- Copy webhook URL
- Paste into n8n HTTP Request node

## Step 3: Set Schedule
- Define trigger interval (e.g. 1h / 6h / daily)

## Step 4: Activate Workflow
- Enable workflow in n8n
